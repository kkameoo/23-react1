### 4.장점
    - 빠른 업데이트와 렌더링 속도
        DOM 동기식 <-> Virtual DOM 비동기식
        DOM - XML,HTML 각 문서의 항목을 계층으로 표현해 생성,변형,삭제할 수 있도록 돕는 인터페이스
        Virtual DOM - DOM의 조작을 효율적으로 만들어 속도를 향상시킨 방법
### 5.컴포넌트 기반 구조
